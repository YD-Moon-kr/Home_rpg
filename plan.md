# ⚔️ 집안일 RPG 게임 개발 계획서 (Development Plan)

## 📌 개요
집안일을 완료하여 경험치(XP)와 골드를 획득하고 레벨업하는 "집안일 RPG" 게임의 프론트엔드 웹 애플리케이션 개발 계획입니다. 기존 콘솔 기반 프로토타입을 화려하고 직관적인 현대적 대시보드 UI/UX로 재구축합니다.

---

## 📅 단계별 로드맵 (Roadmap)

### 🚀 Phase 1: 현대적 대시보드 UI/UX 구축 및 캐릭터 상태 시각화 (현재 진행 단계)
- **목표**: 세련되고 직관적인 게임 분위기의 메인 대시보드 및 세분화/반복 가능한 퀘스트 시스템 구축.
- **주요 작업**:
  - [x] Google Fonts (Outfit & Inter) 도입 및 글로벌 디자인 시스템 (CSS 변수, 다크/네온 테마) 정의.
  - [x] 반응형 Grid 레이아웃 설계 (캐릭터 상태 영역, 퀘스트 목록 영역, 냉장고 인벤토리 영역).
  - [x] 캐릭터 프로필 시각화 (레벨, 경험치 프로그레시브 바, 골드 보유량 네온 카드).
  - [x] 세분화된 하위 퀘스트(Sub-quests) 구현 및 상위 퀘스트 클릭 시 아코디언 형태로 펼쳐지는 UI.
  - [x] 퀘스트 및 하위 퀘스트 반복(Repeatable) 가능하도록 리셋/다시하기 기능 제공.
  - [x] 브라우저 로컬 스토리지 자동 저장/불러오기 기능 검증.


### 🧊 Phase 2: 냉장고 던전 (식재료 관리 시스템) UI/UX 구현
- **목표**: 냉장고 안 식재료의 유통기한을 시각적으로 모니터링하고 관리하는 인벤토리 시스템 구현.
- **주요 작업**:
  - [x] 냉장고 보관 품목 리스트를 게임 인벤토리 그리드 스타일로 렌더링.
  - [x] 식재료 추가 폼/모달 구현 (이름, 카테고리별 유통기한 자동 계산).
  - [x] 유통기한 상태에 따른 시각적 경고 시스템 (위험/경고/안전 색상 및 배지 적용).
  - [x] 식재료 소비("먹기") 및 "폐기" 액션 버튼과 이에 따른 경험치/골드 획득 연동.

### ⚔️ Phase 3: 커스텀 퀘스트 관리 및 게임 플레이 강화 (현재 진행 단계)
- **목표**: 고정된 퀘스트를 넘어 사용자가 직접 집안일 퀘스트를 생성하고 관리할 수 있도록 기능 확장 및 시각 요소 강화.
- **주요 작업**:
  - [x] 커스텀 집안일 퀘스트 추가/삭제 기능 (난이도 선택 및 난이도별 XP/Gold 자동 책정).
  - [x] 퀘스트 완료 시 파티클/페이드 애니메이션 효과 적용.
  - [x] 레벨업 시 화려한 모달 이펙트 구현.
  - [x] AI 이미지 생성기를 활용한 캐릭터 아바타 및 퀘스트별 커스텀 그래픽 일러스트/아이콘 도입.

### 📱 Phase 4: 모바일 최적화 및 최종 폴리싱 & RPG 상점 (진행 예정)
- **목표**: 모바일 기기 반응형 최적화와 최종 폴리싱, 그리고 획득한 골드를 소비할 수 있는 RPG 상점 시스템 및 PWA 연동.
- **주요 작업**:
  - [x] 데이터 초기화/리셋 기능 추가 및 헤더 배치 완료.
  - [ ] 모바일 터치 디바이스 인터랙션 보완 (모바일 뷰에서 팝오버 짤림 방지 및 터치 타겟 최적화).
  - [ ] **골드 소모처: 냐옹이 꾸미기 상점(Shop) 시스템 구현** (골드로 장비/가구를 구매해 냐옹이를 꾸미거나 스탯 버프 획득).
  - [ ] **PWA (Progressive Web App) 설정** (모바일 기기 홈 화면에 앱처럼 설치하여 오프라인 작동 지원).
